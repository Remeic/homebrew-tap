# homebrew-tap

Official Homebrew tap for Remeic command-line tools.

## Install

```sh
brew tap remeic/tap
brew install remeic/tap/ccm
```

Or install directly without a separate tap step:

```sh
brew install remeic/tap/ccm
```

## Available Formulae

### `ccm`

Multi-profile manager for Claude Code.

```sh
ccm --version
ccm list
```

## Maintenance

This tap is updated from the main project release workflow in [Remeic/ccm](https://github.com/Remeic/ccm).

Formula updates are generated from the published npm package to keep the Homebrew install path reproducible and aligned with released versions.
